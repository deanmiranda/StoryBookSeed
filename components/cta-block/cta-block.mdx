**Sitecore Implementation Instructions**

**1.** Run `yarn build`, then check the `dist` folder for the assets you need to copy over.

**2.** For the CTA block variation with the image on the _left side_, add an `.is-reversed` class to the container. For example:

```html
<article class="cta-block is-reversed" data-id="cta-block">
    <!-- etc -->
</article>
```
